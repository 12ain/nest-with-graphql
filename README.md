# Nest Project Template With GraphQL

## Structure

```bash
tree -d -I node_modules
```

```
.
├── bin                         # cli/scripts
├── src                         # source folder
│   ├── config                  # config files
│   ├── decorators              # decorators, `guards`/`exceptions`/`pipes`... if needed
│   ├── dto                     # defined input types
│   ├── models                  # defined database model types
│   ├── modules                 # modules
│   ├── resolvers               # resolvers
│   └── services                # services
└── test                        # test

```
